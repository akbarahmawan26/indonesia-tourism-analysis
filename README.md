## AI Specialist Portfolio Project for UKP Pariwisata

This repository showcases data analysis and visualization skills applied to Indonesian tourism data, as part of preparation for the AI Specialist role at UKP Pariwisata.

---

## 📊 Project Overview

**Objective:** Analyze Indonesian tourism patterns to identify trends, satisfaction metrics, and revenue insights for data-driven decision making.

**Dataset:** 200 tourism records across 8 major Indonesian destinations (Bali, Yogyakarta, Jakarta, Bandung, Surabaya, Medan, Makassar, Lombok) for 2024.

**Key Metrics Analyzed:**
- Visitor trends by destination and month
- Revenue distribution (Domestic vs International)
- Tourist satisfaction scores
- Peak season identification

---

## 🛠️ Technologies Used

- **Python 3.11** - Data processing and analysis
- **Pandas** - Data manipulation and aggregation
- **Matplotlib & Seaborn** - Data visualization
- **Jupyter-ready** - For interactive exploration

---

## 📈 Key Findings

### Visitor Distribution
- **Bali** leads with **2.1M visitors** (most popular destination)
- **Domestic tourists:** 8.4M (75.6%)
- **International tourists:** 2.7M (24.4%)

### Revenue Analysis
- **Bandung** generated highest revenue: **2.9 Trillion IDR**
- Average spend per international visitor: **3.2M IDR**
- Average spend per domestic visitor: **992K IDR**

### Satisfaction Scores
- **Bandung:** 8.1/10 (highest satisfaction)
- **Jakarta:** 8.0/10
- Overall average: **7.87/10**

### Seasonal Trends
- **Peak months:** March, January, May
- **Opportunity:** July-August could be promoted more
